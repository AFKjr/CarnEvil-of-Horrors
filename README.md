# CarnEvil of Horrors

A spooky Halloween-themed web project featuring horror effects and animations.

## Description

This project creates an interactive horror-themed carnival experience with various spooky visual effects including screen flickers, glitch animations, blood drips, and other eerie elements.

## Files

- `Spooky_Challenge.html` - Main HTML file with the horror carnival layout
- `effects.js` - JavaScript file containing spooky effects and animations

## Features

- Random screen flicker effects
- Glitch animations on text elements
- Blood drip visual effects
- Horror-themed color scheme (blood red, toxic green, bone white, shadow purple)
- Responsive design
- HTML5
- CSS3 (with custom properties/variables)
- JavaScript (vanilla)
